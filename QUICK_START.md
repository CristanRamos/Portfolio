### 1. View Your Portfolio

### 2. Customize (Optional)
You can customize the following:

**Personal Information:**
- `components/sections/Hero.tsx` - Update name, title, or description
- `components/Footer.tsx` - Update contact details
- `components/sections/About.tsx` - Modify education or achievements

**Projects:**
- `components/sections/Projects.tsx` - Add GitHub links or demo URLs
- Update project descriptions or add new projects

**Styling:**
- `tailwind.config.js` - Change color theme
- `app/globals.css` - Modify global styles
